# Space Harbour

**Title:** Space Harbour  
**Authors:** Andrei Romashkov  

## Description
Space Harbour is a 2D platformer game where the player defends a Space Harbour from invading aliens. Player can run, jump, and shoot weapons while navigating platforms and completing levels.

## Purpose of the program
A fun, interactive game to practice programming in Python with Pygame, demonstrating game mechanics, collision detection, shooting, and level progression.
